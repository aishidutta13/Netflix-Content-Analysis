# Netflix Content Analysis

## Project Overview

This project analyzes Netflix’s global content dataset to understand content growth trends, geographical distribution, and audience targeting patterns. The focus is on building a structured analytical workflow and presenting the results through a dashboard.

---

## Objective

To explore Netflix content data and identify patterns in:

* Content growth over time
* Regional distribution of titles
* Audience targeting through content ratings

---

## Dataset

* Source: Netflix Titles Dataset
* Format: CSV
* Key columns:

  * Title
  * Country
  * Release Year
  * Date Added
  * Rating
  * Duration
  * Genre

---

## Tools and Technologies

* Python (Pandas, NumPy)
* Jupyter Notebook (Data Cleaning and Analysis)
* IBM Cognos Analytics (Dashboard Development)

---

## Methodology

### Data Cleaning

* Handled missing values
* Processed categorical columns
* Converted date fields
* Standardized column formats

### Data Preparation

* Extracted year from date_added
* Aggregated data for trend analysis
* Processed country data for mapping

### Visualization

Developed a dashboard consisting of:

* Global content distribution (map)
* Content distribution by rating (bar chart)
* Content growth over time (line chart)

---

## Dashboard Description

### Global Distribution of Netflix Content

Displays the number of titles available across countries, highlighting geographical spread.

### Content Distribution by Rating

Shows how Netflix content is distributed across different audience categories.

### Netflix Content Growth Over Time

Tracks how content additions have evolved over time, indicating growth patterns.
